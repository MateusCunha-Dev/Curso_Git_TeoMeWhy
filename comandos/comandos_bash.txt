GIT - Aula 02
ls = lista o que existe no diretório
cd = muda de diretório
pwd = caminho absoluto de onde estou
ls -a = arquivos oculto 
cd .. = Volta para o diretório pai (um nível acima)
cat "arquivo" = para visualizar o conteúdo
rm = remove arquivo do diretorio
nano "nome_arquivo" entra no editor de texto nano
ctrl o = salva alteração no arquivo
ctrl x = sai do editor nano
rm -rf = remove de forma recursiva e forçada todos os diretórios (muito cuidado)
